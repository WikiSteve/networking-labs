# Catalyst 1300 Tour

## Goal
Explore the physical and logical configuration of a Cisco Catalyst 1300 Series switch, including basic IP setup, DHCP, SSH, and inter-VLAN routing.

## Prerequisites
- Physical access to a Catalyst 1300 switch
- Console cable and terminal emulator (e.g., PuTTY, Tera Term, or `screen`)
- Basic understanding of CLI and networking concepts

## Deliverables
- Completed configuration of management IP and services
- Verification screenshots as requested in individual lab pages
- Functional inter-VLAN routing setup
