# Groups exercise

## Add the Following Groups

`managers`, `executive`, `hr`, `it`, `sales`, and `users`

## Add the Following Users

`sally`, `fred`, `jane`, `bob`, `sam`, `mary`, `bill`, `larry`, `tammy`, and `lisa`

> [!NOTE]
> Use the same password for all of these users so you can compare the resulting password hashes later.

---
### 🧠 Password reuse

> [!NOTE]
> **Statement:** Because all users were given the same password, the stored hashes must be identical.
>
> - [ ] True
> - [ ] False

<details>
<summary>👉 <b>Reveal answer</b></summary>

**Correct:** False
</details>

---

### Practice check

Even when two users reuse the same password, their stored hashes can differ because the hashes are salted.
