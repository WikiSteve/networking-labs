# Lecture Linux Users, Groups, and Mode

## Goal
Review Linux users, groups, password storage, and file permissions, including the special permission bits `setuid`, `setgid`, and `sticky`.

## Prereqs
- Basic Linux command-line familiarity
- Access to a Linux system or virtual machine for trying commands such as `cat`, `ls -l`, `chmod`, `chown`, and `adduser`
- Comfort reading `/etc/passwd`, `/etc/group`, and `/etc/shadow` at a high level

## Deliverables
- Work through the lecture notes, practice checks, and linked walkthrough videos
- Complete any follow-up exercises your instructor assigns from the users, groups, or mode sections
