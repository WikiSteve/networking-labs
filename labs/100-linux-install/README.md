# Linux Install

## Goal
Download a Debian netinst ISO, build a Debian virtual machine in VMware Workstation, start the installer, and manually partition the disk with LVM.

## Prereqs
- VMware Workstation or a similar virtualization platform
- Enough local storage for a Debian virtual machine and installer ISO
- Internet access to download the Debian netinst image

## Deliverables
- Screenshot 1: your completed Debian VM hardware profile
- Screenshot 2: `lvdisplay | less` showing `lv-swap`, `lv-home`, and `lv-root`
