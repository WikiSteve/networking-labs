# Windows 11 Software

## Additional Required Software

- **PuTTY**: A terminal emulator for SSH connections. It is especially useful when you need X11 forwarding from a remote Linux system. [Download PuTTY](https://www.putty.org/)
- **VcXsrv**: An X11 display server for Windows that lets graphical Linux programs display locally over an SSH session. [VcXsrv Windows X Server](https://sourceforge.net/projects/vcxsrv/)
- **Wireshark**: A packet analyzer that helps students inspect and troubleshoot network traffic in later labs. [Download Wireshark](https://www.wireshark.org/)
