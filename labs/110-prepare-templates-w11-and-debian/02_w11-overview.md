# Windows 11 Overview

## Windows 11 Template Guide

In this setup guide, you’ll prepare a **Windows 11 Template** that will serve as the foundation for future practicals. Each step will guide you through configuring the system name, creating and verifying a local administrator user, setting up the date and time, and customizing the PowerShell prompt for consistency across sessions.

**Important reminder**: Unless a page explicitly says otherwise, perform the Windows steps in **PowerShell ISE opened as Administrator**.

## Start PowerShell ISE as Administrator

1. Press **Windows Key + S** to open the search bar.
2. Type **PowerShell ISE**.
3. Right-click **Windows PowerShell ISE** in the search results.
4. Select **Run as Administrator**.

Following these steps will provide the environment needed to complete the setup successfully. Once in PowerShell ISE, carefully follow each section, verifying each configuration as you progress. This approach ensures that every setting is correctly applied and prepares you for a smooth experience in the upcoming practicals.
