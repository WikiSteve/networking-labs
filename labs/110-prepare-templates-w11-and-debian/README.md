# Lab Prepare Templates W11 and Debian

## Goal
Prepare reusable Windows 11 and Debian virtual machine templates by cloning the base VMs, applying the required configuration, and installing the supporting tools used in later labs.

## Prereqs
- Access to the Windows 11 and Debian VMs in VMware Workstation
- Administrator access on Windows and `sudo` access on Debian
- Internet access for required downloads such as OpenVPN, PuTTY, VcXsrv, and Wireshark

## Deliverables
- Complete the required configuration steps for both templates
- Capture the screenshots or verification output your instructor asks for before the templates are reused in later labs
