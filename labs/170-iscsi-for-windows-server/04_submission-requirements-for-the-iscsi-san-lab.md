# Submission Requirements for the iSCSI SAN Lab

## General rules

- show the computer name in every required screenshot
- capture only the relevant area of the screen
- make sure the text is legible
- submit only the exact screenshots requested by the lab

## Showing the computer name

On Windows, open Command Prompt and run:

```bat
echo %COMPUTERNAME%
```

Leave that visible in the screenshot whenever possible so it is clear which server performed the task.

## Submission format

- use the PowerPoint submission template for this lab
- place the screenshots in the same order as the lab steps
- keep any short captions or notes brief and specific
- name the finished file `Lastname_Firstname_iSCSILab.pptx`
