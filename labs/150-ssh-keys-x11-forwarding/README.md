# SSH Keys and X11 Forwarding

## Goal
Practice verifying an SSH server key, authenticating with user key pairs in PuTTY, loading multiple keys with Pageant, and forwarding X11 applications from Debian to Windows.

## Prereqs
- A Debian lab VM with `openssh-server` installed and reachable from Windows
- A Windows 11 VM with PuTTY available
- Basic comfort with Linux shell commands, file paths, and SSH logins

## Deliverables
- Complete the four required screenshots
- Use the submission template your instructor provides for this lab
- Show that you can verify the server key, authenticate with SSH keys, use Pageant, and launch `xterm` over X11 forwarding
