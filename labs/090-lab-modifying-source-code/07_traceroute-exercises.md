# Traceroute Exercises

Clear your screen and then work through the following traceroute tasks.

## Screenshot 7

Screen print a working traceroute to `www.google.ca`, including the directory and command you used to run it.

Then complete the following:

- Run a standard traceroute to `www.google.ca` limited to 5 hops.
- Search the man page to find the options that force the use of ICMP packets and IPv4, then run traceroute again.
- Search the man page to find the options that force the use of TCP packets and set the source IP address to your second network address, `192.168.92.1`.

> [!TIP]
> The source material intentionally leaves the exact command options for you to discover with the man page.
