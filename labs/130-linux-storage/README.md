# Linux Storage

> **Before you start:** Download the [Linux Storage Submission Template](<./assets/Linux Storage SUBMISSION TEMPLATE.pptx>). Add each required screenshot directly into this file as you complete the lab, then submit the completed template for grading.

## Goal
Practice common Linux storage administration tasks with LVM, including removing the `sudo` password prompt for the lab VM, resizing logical volumes, migrating data to a new drive, reinstalling GRUB, and restoring files from a snapshot.

## Prereqs
- A working Debian lab VM from the earlier Linux labs
- Basic comfort with `sudo`, `df`, `lsblk`, `mount`, and simple text editing
- VMware Workstation access for adding and removing virtual disks
- Familiarity with the idea of physical volumes, volume groups, and logical volumes in LVM

## Deliverables
- Complete the five required screenshots from the lab
- Finish the quiz and PowerPoint submission your instructor requires
- Make sure the screenshots are readable and highlighted as requested in the lab steps
