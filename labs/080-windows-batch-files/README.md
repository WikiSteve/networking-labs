# Lab Windows Batch Files

## Goal
Use Windows command-line tools to create links, build simple batch files, work with `for` loops, and observe how scripts can automate tasks or disguise program behavior.

## Prereqs
- A Windows 10 virtual machine with both standard and administrator access
- Lab files available under `\info1218`, including `felix.exe` and the `sysinternals` folder
- VMware Workstation configured to use NAT networking when required

## Deliverables
- Complete the required command output and screenshots for each section
- Submit your work using the course submission template
