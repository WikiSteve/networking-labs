# SSH and MAC Operations Guide

## Goal
Set up SSH key-based access with PuTTY and GitHub, then review the core Linux file and directory commands used for move, add, and change work.

## Prerequisites
- A Windows system with PuTTY and PuTTYgen installed
- Access to two Linux systems such as Ubuntu and CentOS/RHEL
- A GitHub account
- Basic comfort with the terminal on both Windows and Linux
